# 💣 Bomb Party

## 📌 Overview

A simple Bomb party Game made in Javascript.

## 🔧 Technologies

- Typescript
- React.js
- Socket.io
- ESLint
- Express

## 🚀 Getting started

1. Clone this repository
2. Run the following command

```sh
# ./frontend/
yarn install
yarn dev

# ./backend/
yarn install
yarn dev
```

## 🏛️ Architecture Backend

```
# Environnement
./.env.development.local
./.env.production.local
./.env.test.local


# Configuration - All files that correspond to configuration
/src/config


# Constants - All files that have constants
/src/constants


# Controllers - That directly responds to each request that comes into the application
/src/controllers


# Exceptions - All exceptions
/src/exceptions


# Interfaces - All types interfaces
/src/interfaces


# Middlewares - All middlewares
/src/middlewares


# Routes - All routes
/src/routes


# Services - All services
/src/services

```
